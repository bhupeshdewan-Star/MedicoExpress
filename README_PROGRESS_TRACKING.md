## 🟢 CONFIDENCE INDICATORS

### How to tell if we're on track

**Green Flag (On Track):**
- Task % increases 10-15% per day
- Gates passing on schedule
- No blockers in Slack
- Daily standups completed
- Slack updates posted consistently

**Yellow Flag (Monitor):**
- Task % increases only 5% per day
- One gate delayed 1 day
- Minor blocker resolved same day
- Team stretched (95%+ utilization)
- Resource requests made

**Red Flag (Escalate):**
- Task % stays flat (no change in 24h)
- Gate fails and goes to rework
- Blocker open >4 hours without fix
- Team unavailable (emergency absence)
- Escalation to VP/CMO made
