## 🔴 RED FLAGS (What to Worry About)

### Warning Signs in Percentages

**If you see this:** → **It means this:**

| Percentage | Indicator | What to Do |
|-----------|-----------|-----------|
| **0% at Day 3** | Task not started (should be 30%+) | 🚨 ESCALATE IMMEDIATELY |
| **Dropping %** | Work being redone/deleted | 🚨 CHECK FOR BLOCKER |
| **Stuck at same %** | No progress made | ⚠️ INVESTIGATE CAUSE |
| **Slower than expected** | Behind schedule | 🟡 MONITOR CLOSELY |
| **On track** | 5-10% per day, on pace | ✅ CONTINUE EXECUTION |

### Normal vs Concerning Pace

```
TASK 6 (5-day task):
✅ NORMAL:  5% → 10% → 25% → 50% → 100% (accelerating)
✅ NORMAL:  5% → 15% → 30% → 60% → 100% (steady)
⚠️  SLOW:   5% → 10% → 15% → 20% → 100% (catching up at end)
🚨 BLOCKED: 5% → 5% → 5% → 5% → ESCALATE
```
