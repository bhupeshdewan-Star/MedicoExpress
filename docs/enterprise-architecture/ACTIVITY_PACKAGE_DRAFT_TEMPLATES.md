## 4. Draft Control Checklist Template

```text
Critical checks:
- Any failure blocks progression.

Major checks:
- Must be resolved or formally dispositioned.

Minor checks:
- Editorial or non-material issues.

Approval meaning:
- Exact statement signed by the accountable approver.

Revision triggers:
- Source, safety, label, regulatory, SOP, intended-use, or output changes.
```

## 5. Review Cover Sheet

Every ClinCommand-proposed draft should be presented with:

- source documents consulted;
- assumptions made;
- missing owner inputs;
- proposed additions;
- conflicts requiring decision;
- implementation impact;
- requested owner decisions;
- explicit statement that the draft is not effective or finalized.
