# Skill Certification Rubric — Gate 7.0 Foundation Directive

## 1. Quality Certification Categories
Each skill in the repository is graded at runtime and mapped to a certification category:
- **95–100**: World-Class
- **90–94**: Enterprise Grade
- **80–89**: Production Grade
- **70–79**: Needs Improvement
- **Below 70**: Reject (Blocked from execution)

## 2. Evaluation Dimensions & Weighted Scoring
The Dynamic Quality Engine scores outputs across 8 distinct GxP dimensions:
1. **Domain Expertise** (Weight: 20%) — Depth of scientific, clinical, and regulatory terminology.
2. **Output Structure** (Weight: 15%) — Compliance with structured markdown headers and tabular layout requirements.
3. **Compliance** (Weight: 15%) — Conformance with regulatory guidelines (FDA, EMA, 21 CFR Part 11).
4. **Explainability** (Weight: 10%) — Integrity of purpose, assumptions, limitations, and source references.
5. **Governance** (Weight: 10%) — SOP linkage, review dates, and metadata attribution checks.
6. **Evidence Quality** (Weight: 10%) — Grounding assertions in statistical trial metrics (p-values, trial cohorts).
7. **Practical Utility** (Weight: 10%) — Strategic value and actionability of recommendations.
8. **Reusability** (Weight: 10%) — Generalizability across trials, products, and users.
